Tasks

Turn scripts into
production-grade apps

myTask.spark.ts 

import { refund } from "../lib/businessLogic";

export default spark.task(
    { slug: "my_task" },
    (params) => {
    const txID = refund(
        params.userID, params.amount
    );
    return { "confirmation": txID };
    },
);

Tasks are scripts for your entire team.

Stop running backfills from your laptop. Deploy JS, Python, SQL, and more to spark and get production-grade apps.
